# Recommended Tools — macOS

## Prerequisites

### Homebrew

[Homebrew](https://brew.sh/) is the most popular package manager for macOS. To install it, open Terminal and run:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

After installation, follow the on-screen instructions to add `brew` to your PATH. You can verify the installation by running:

```bash
brew --version
```

## Tools

[Oh-my-Posh](https://ohmyposh.dev/) — A prompt theme engine for any shell

```bash
brew install jandedobbeleer/oh-my-posh/oh-my-posh
```

[Atuin](https://atuin.sh/) — Magical shell history search & sync

```bash
brew install atuin
```

[Visual Studio Code](https://code.visualstudio.com/) — Lightweight, powerful code editor by Microsoft

```bash
brew install --cask visual-studio-code
```

[Git](https://git-scm.com/) — Distributed version control system

```bash
brew install git
```
